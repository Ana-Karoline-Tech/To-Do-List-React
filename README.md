# 📝 To-Do List React

Um aplicativo de lista de tarefas moderno e intuitivo desenvolvido com **React**, oferecendo uma forma simples e eficiente de gerenciar suas atividades diárias.

## ✨ Funcionalidades

- ✅ Adicionar novas tarefas
- ❌ Marcar tarefas como concluídas
- 🗑️ Deletar tarefas
- 📋 Visualizar todas as tarefas em tempo real
- 💾 Interface limpa e responsiva

## 🚀 Como Usar

### Pré-requisitos

- Node.js (versão 14 ou superior)
- npm ou yarn

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/Ana-Karoline-Tech/To-Do-List-React.git
cd To-Do-List-React
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm start
```

4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver a aplicação rodando.
